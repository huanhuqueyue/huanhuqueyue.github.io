Do We Really Need to Access the Source Data? Source Hypothesis Transfer for Unsupervised Domain Adaptation

最近终于找到生活的节奏了，一直忙碌，忙碌起来就不会伤心难过了，事情还是要往好的地方想，每晚十点五十从工作室狂奔回寝室，没有比这更快乐的了，虽然造化弄人8，我大学持续时间最长的愿望落空了。所有的事情可能都要怪我8，没有看到生活背后的一些如金子般闪光的东西。但我相信，有些人有些事有些爱，注定要羁绊很久，就注定像一棵树一样，生长在心里，生根发芽。

好了，我们回到这篇文章的主题：

UDA 无监督DA 是通过把一个领域的知识（有标签可以监督的）迁移到另外一个领域中

传统方法有什么缺点？

需要在adaptive的时候接触到源领域的数据，数据构建起一个联合的数据分布，这可能会对个人隐私产生一定的影响

如何解决？

只用训练模型，不用训练数据，完成DA的transfer

具体方法：

一个简单通用的表达：shot

固定预测的模块，增加一个函数和目标相关的特征提取模块

特征提取模块有啥用？

  • 信息最大化
  • 自监督的伪标签学习 来完成alignment的任务（不会很充分）

有什么优点：

  • 保密性高
  • 和数据相比，更轻量级
